Welcome to Esperto Labs!

We are a brand new design team from the University of Waterloo that aims to design a wearable platform easy enough for beginners to learn with but also complex and powerful enough for researchers and developers to use in their work.

Sensors

An accelerometer, gyroscope, heart rate sensor, sp02 and a temperature sensor are integrated into the Esperto Watch.

Wireless Connectivity

The Esperto watch supports wireless connectivity through bluetooth low energy.

Companion Applications

We've created a web and mobile application allowing for visualization and analysis of data obtained through the Esperto watch.

Display

The Esperto watch features a low power OLED that is easy to use for development purposes.

Programmability

The Esperto watch features memory, making it possible to implement your own custom software.

Customizability

The Esperto watch uses open source hardware, and features enough space to implement new hardware.
